# rad-to-turn-cli

**rad-to-turn-cli** is a CLI tool that converts radians to turns.

![npm](https://img.shields.io/npm/v/rad-to-turn-cli)
![license](https://img.shields.io/npm/l/rad-to-turn-cli)

## Quick Start

Get **rad-to-turn-cli** up and running quickly!

1. **Run**:
    ```bash
    npx rad-to-turn-cli 3.14159
    ```
2. **Done!**

### Example Output

This will convert the radians value `3.14159 rad` to its turns equivalent and display the result in your terminal.

- **Input**: `3.14159`
- **Output**: `Turns: 0.5 turn(s)`
